# FocusTube

FocusTube is a Chrome extension that removes the YouTube homepage and recommendations, leaving only the search bar so you can search intentionally without distractions.

## Features

- ✅ Hides the YouTube homepage
- ✅ Removes recommended videos
- ✅ Keeps only the search bar
- ✅ Lightweight and fast

## Installation

1. Download this repository:
   - Click **Code → Download ZIP**
2. Extract the ZIP.
3. Open Chrome.
4. Go to `chrome://extensions`
5. Enable **Developer mode**.
6. Click **Load unpacked**.
7. Select the extracted folder.

Enjoy distraction-free YouTube!
